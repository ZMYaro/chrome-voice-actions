“OK Chrome” for Voice Actions for Chrome


1 Install Voice Actions for Chrome

“OK Chrome” is an add-on for the Voice Actions for Chrome extension. You must have Voice Actions for Chrome installed and enabled for “OK Chrome” to activate it

Install Voice Actions

2 Grant microphone access

“OK Chrome” needs permission to access your microphone for speech recognition. After you select the button to show the permission dialog, select “Allow” in the dialog that appears by the Chrome toolbar. If you don't see the permission dialog, you might need to allow microphone access in Chrome settings.

Also, make sure your microphone is connected and turned on.


3 Activate the extension

“OK Chrome” is a premium feature for Voice Actions for Chrome. Select the button to activate. If you already activated “OK Chrome” on another device and are being charged again, make sure you are signed into Chrome with the same account.

“OK Chrome” for Voice Actions for Chrome


1 You must have Voice Actions for Chrome installed

This is an add-on for the Voice Actions for Chrome extension. You must have Voice Actions for Chrome installed and enabled for “OK Chrome” to activate it.


2 The extension needs permission to access your microphone for speech recognition.

Select “Allow” in the dialog above. If you don't see the permission dialog, you might need to allow microphone access in Chrome settings.

Also, make sure your microphone is connected and turned on.


3 Sign in with Ko-fi or Patreon

“OK Chrome” activation is a premium feature for Voice Actions for Chrome for supporters. Sign in with Ko-fi or Patreon to activate it.

Help

For speech recognition to work, you need to allow microphone access in Chrome settings.

Make sure your microphone is connected and turned on.

“OK Chrome” is an add-on for the Voice Actions for Chrome extension. You can uninstall “OK Chrome” for Voice Actions for Chrome to disable the “OK Chrome” hotword and still use the rest of Voice Actions for Chrome.
